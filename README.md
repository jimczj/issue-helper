# issue-helper

## 简介

根据配置文件生成 github issue 提交表单

## 下载安装

```shell
npm i -g issue-helper
```

## 生成默认配置 config.js

```shell
issue-helper init
```

## 根据需求配置 config.js

## 预览页面

在 config.js 同级目录下执行命令，页面会根据 config.js 的更改热更新。

```shell
issue-helper preview
```

## 编译生成 issue helper 静态页面

在 config.js 同级目录下执行命令

```shell
issue-helper build

```
